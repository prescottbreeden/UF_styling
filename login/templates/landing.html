Welcome to Ultimate Fan!

Description:

Ultimate Fan is a trivia game that pits users against each other to see who can answer the most questions about athletes from selected categories.

Instuctions:

To play, please register by clicking the button below or by signing in. Select a category and questions will populate one at a time to be answered. Questions are timed and if the bar below the question reaches 0 the question will be counted as a wrong answer before giving the next question.

About:

All data for the trivia questions were scraped using beautiful soup.

Technologies used: Python, Django, SQLite, bcrypt, CSS3, HTML5, SASS, jQuery, AWS

This project was made by Prescott Breeden and Stephen Silverstein during our python stack at Coding Dojo.
